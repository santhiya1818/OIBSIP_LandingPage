# Lumara - Creative Studio Landing Page

This is my submission for the **Level 1 Task 1** of the Oasis Infobyte Web Development Internship.

## Project Description
A responsive landing page for "Lumara," a boutique design studio. The page features a modern, aesthetic design with a hero section, services grid, portfolio gallery, testimonials, and a contact form. It demonstrates proficiency in HTML5, CSS3 (Flexbox & Grid), and basic JavaScript for interactivity.

## Technologies Used
- HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (Scroll animations, Form interaction)
